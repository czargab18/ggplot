Grfico de Barras ou Colunas Agrupadas:

Variveis: Qualitativas (categricas) e Quantitativas. Uso: Compara valores quantitativos para diferentes categorias qualitativas.

# Instale o pacote ggplot2 se ainda no estiver instalado
#  install.packages("tidyverse")

# Carregue o pacote ggplot2
library(ggplot2)

# Crie um conjunto de dados fictcio
dados <- data.frame(
  Categoria = rep(c("A", "B", "C"), each = 4),
  Valor = c(10, 15, 8, 12, 14, 18, 9, 13, 11, 16, 7, 10)
)

# Crie o grfico de barras agrupadas
ggplot(dados, aes(x = Categoria, y = Valor, fill = Categoria)) +
  geom_bar(stat = "identity", position = "dodge") +
  labs(title = "Grfico de Barras Agrupadas",
       x = "Categoria",
       y = "Valor") +
  theme_minimal()

# Crie um conjunto de dados fictcio com duas sries

dados_long <- tidyr::gather(dados, key = "Serie", value = "Valor", -Categoria)

  geom_col(position = "dodge", width = 0.7) +
  labs(title = "Grfico de Colunas Agrupadas",
       x = "Categoria",
       y = "Valor",
       fill = "Srie") +
  theme_minimal()

Box Plot (Diagrama de Caixa):

Variveis: Qualitativas (categricas) e Quantitativas. Uso: Representa a distribuio estatstica dos dados, destacando a mediana, quartis e outliers.

# Instale o pacote ggplot2 se ainda no estiver instalado
# install.packages("ggplot2")

# Carregue o pacote ggplot2
library(ggplot2)

# Crie um conjunto de dados fictcio
dados <- data.frame(
  Categoria = rep(c("A", "B", "C"), each = 50),
  Valor = c(rnorm(50, mean = 10, sd = 2),
            rnorm(50, mean = 15, sd = 2),
            rnorm(50, mean = 12, sd = 3))
)

# Crie o box plot
ggplot(dados, aes(x = Categoria, y = Valor, fill = Categoria)) +
  geom_boxplot() +
  labs(title = "Box Plot (Diagrama de Caixa)",
       x = "Categoria",
       y = "Valor") +
  theme_minimal()

Grfico de Bolhas:

Variveis: Duas variveis quantitativas e uma qualitativa (tamanho das bolhas). Uso: Mostra relaes entre trs variveis, incluindo o tamanho das bolhas para representar uma terceira dimenso.

# Instale o pacote ggplot2 se ainda no estiver instalado
# install.packages("ggplot2")

# Carregue o pacote ggplot2
library(ggplot2)

# Crie um conjunto de dados fictcio
dados <- data.frame(
  VariavelX = c(10, 15, 20, 25),
  VariavelY = c(5, 10, 15, 20),
  TamanhoBolha = c(30, 60, 90, 120),
  Categoria = c("A", "B", "C", "D")
)

# Crie o grfico de bolhas
ggplot(dados, aes(x = VariavelX, y = VariavelY, size = TamanhoBolha, color = Categoria)) +
  geom_point(alpha = 0.7) +
  scale_size_continuous(range = c(3, 15)) +
  labs(title = "Grfico de Bolhas",
       x = "Varivel X",
       y = "Varivel Y",
       size = "Tamanho da Bolha",
       color = "Categoria") +
  theme_minimal()

Grfico de Radar ou Teia (Spider Plot):

Variveis: Qualitativas (categorias) e Quantitativas. Uso: Comparao de vrias categorias em termos de mltiplas variveis quantitativas.

# Instale o pacote ggplot2 se ainda no estiver instalado
# install.packages("ggplot2")

# Carregue o pacote ggplot2
library(ggplot2)

# Crie um conjunto de dados fictcio
dados <- data.frame(
  Categoria = c("A", "B", "C", "D"),
  Variavel1 = c(5, 8, 12, 15),
  Variavel2 = c(10, 15, 18, 22),
  Variavel3 = c(7, 12, 14, 20)
)

# Crie o grfico de radar
ggplot(dados, aes(x = Categoria)) +
  geom_polygon(aes(y = Variavel1, group = 1), fill = "lightblue", alpha = 0.5) +
  geom_polygon(aes(y = Variavel2, group = 2), fill = "lightgreen", alpha = 0.5) +
  geom_polygon(aes(y = Variavel3, group = 3), fill = "lightcoral", alpha = 0.5) +
  labs(title = "Grfico de Radar",
       y = "Valor",
       x = "Categoria") +
  theme_minimal()

Variveis Unidimensionais Quantitativas (Univariadas):

Exemplos: Idade, Peso, Altura, Tempo. Grfico Recomendado: Histograma, Grfico de Barras ou Grfico de Linhas. Variveis Qualitativas (Categricas):

Exemplos: Cores, Categorias, Grupos. Grfico Recomendado: Grfico de Barras, Grfico de Pizza (ou Grfico de Setores), Grfico de Pareto. Variveis Bidimensionais (Bivariadas) - Quantitativa x Quantitativa:

Exemplos: Correlao entre duas variveis quantitativas. Grfico Recomendado: Diagrama de Disperso. Variveis Qualitativas Cruzadas (Crosstab):

Exemplos: Tabela de contingncia. Grfico Recomendado: Grfico de Barras Empilhadas, Grfico de Mosaico. Sries Temporais:

Exemplos: Dados ao longo do tempo. Grfico Recomendado: Grfico de Linhas, Grfico de rea. Distribuio de Frequncia:

Exemplos: Frequncia de ocorrncia de valores. Grfico Recomendado: Histograma. Comparao de Percentuais ou Propores:

Exemplos: Partes de um todo. Grfico Recomendado: Grfico de Pizza (ou Grfico de Setores), Grfico de Barras Empilhadas. Comparao entre Grupos:

Exemplos: Comparao entre diferentes categorias. Grfico Recomendado: Grfico de Barras, Grfico de Linhas. Distribuio Normal ou Simtrica:

Exemplos: Dados distribudos normalmente. Grfico Recomendado: Histograma, Grfico de Probabilidade Normal. Variao ao Longo de Categorias:

Exemplos: Variao de uma varivel ao longo de diferentes categorias. Grfico Recomendado: Grfico de Linhas, Grfico de Barras.
